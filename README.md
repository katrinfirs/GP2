# GP2
GP 2 - Scraping &amp; API - team 11 

Проект: Анализ вакансий для студентов бизнес-информатики

Этот проект собирает и анализирует вакансии для студентов бизнес-информатики ВШБ в Москве. Цель - помочь руководству выбрать компании для заключения договоров о практике.

Что делает проект

1. Сбор вакансий с HH.ru
   · Автоматический сбор вакансий через API
   · Сбор вакансий через браузер (Selenium)
   · Фильтрация по профессиям для бизнес-информатики
2. Сбор вакансий с SuperJob
   · Использование официального API
   · Получение актуальных вакансий
3. Обработка данных
   · Объединение данных из разных источников
   · Удаление дубликатов
   · Приведение к единому формату
4. Анализ данных
   · Исследование собранных данных
   · Поиск закономерностей
   · Анализ требований компаний
5. Визуализация
   · Создание графиков и диаграмм
   · Наглядное представление результатов

Для кого этот проект

· Руководство ВШБ - для выбора компаний-партнеров для практики
· Карьерный центр - для помощи в трудоустройстве студентов
· Студенты - для понимания рынка труда

Что можно узнать из анализа

· Какие компании чаще всего нанимают студентов
· Какие навыки самые востребованные
· Уровень зарплат для начинающих специалистов
· Географию вакансий по Москве

Технические требования

· Python 3.8 или новее
· Библиотеки для анализа данных
· Доступ к интернету для сбора вакансий

Как использовать

1. Установить необходимые программы
2. Запустить сбор вакансий
3. Запустить анализ данных
4. Просмотреть результаты в виде графиков и таблиц

Проект помогает автоматизировать поиск компаний-партнеров и принимать обоснованные решения о сотрудничестве.
